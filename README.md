# ☸️ 赛博菩萨 - 国潮赛博风彩票号码生成器

> 寺庙金身 + 霓虹灯管 = 赛博朋克佛教美学 🙏✨

一个融合国潮元素与赛博朋克风格的彩票号码生成器，支持双色球和大乐透。

## 🎨 设计特色

### 国潮赛博风格
- **主色调**：深紫渐变金色（#1a0033 → #ffd700）
- **视觉元素**：寺庙金身 + 霓虹光效 + 玻璃拟态
- **字体搭配**：思源宋体 + Orbitron 科技感字体
- **特效**：
  - Canvas 佛光粒子背景动画
  - 3D 立体号码球（高光+阴影）
  - 发光脉冲按钮
  - 彩虹边框旋转动画
  - 悬浮佛像图标

## 🎮 功能特点

- 🎱 **双色球**: 红球1-33选6 + 蓝球1-16选1
- 🎯 **大乐透**: 前区1-35选5 + 后区1-12选2
- 💰 **智能计算**: 根据金额自动计算可购买注数
- 🎲 **生成动画**: 佛像发光脉冲 + "请示菩萨"仪式
- 📜 **菩萨开示**: 12句随机吉祥语录
- ⭐ **财运评级**: 1-5星动态评级
- 📋 **一键复制**: 快速复制号码
- 📱 **分享功能**: 分享生成结果
- 💾 **历史记录**: 本地保存今日求号记录

## 📱 使用方法

1. 选择彩票类型（双色球/大乐透）
2. 输入供奉金额
3. 点击"请菩萨赐号"
4. 等待佛光普照...
5. 获得天赐号码！
